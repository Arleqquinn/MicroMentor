# MicroMentor

MicroMentor — мобильное приложение для формирования индивидуального плана развивающих заданий в
формате микрообучения.

## Описание проекта

Приложение предназначено для коротких ежедневных занятий и позволяет:

- проходить микроуроки;
- выполнять тестовые задания;
- отслеживать прогресс;
- получать персональный план на день;
- повторять ранее изученный материал;
- накапливать баллы, достижения и streak.

## Основные функции

- регистрация и вход через Firebase Authentication;
- подтверждение email;
- хранение профиля пользователя в Cloud Firestore;
- загрузка курсов и уроков;
- прохождение уроков и квизов;
- сохранение прогресса;
- баллы, уровни, достижения и streak;
- формирование плана на день;
- базовая логика повторения материала.

## Используемые технологии

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- JSONBin.io
- Android Studio

## Структура проекта

- `lib/` — исходный код приложения
- `assets/` — ресурсы приложения
- `android/` — Android-часть проекта
- `lib/config/` — конфигурационные файлы

## Настройка проекта

Перед запуском необходимо создать файл:

`lib/config/secrets.dart`

Пример содержимого:

```dart
